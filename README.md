# Project: Tactical Surveillance Handgrip (TSH-1)

A modular, handheld camera system for video recording and real-time preview. This project was developed to create a stable and professional recording device for mobile operations.

## 🏗️ System Architecture
The system consists of two main subsystems: the self-contained power supply and the active signal processing.

## 🛠️ Component List
* **Camera:** JVC TK 930E (CCTV)
* **Recording:** FPV Mini-DVR
* **Preview:** 4.3" TFT LCD monitor
* **Distribution:** Active video booster (1-to-3)
* **Power:** 12V Li-ion battery with 12V-to-5V step-down converter
* **Mechanics:** Hama tripod pistol grip

## 💡 Technical Challenges (Level 3)
* **Voltage management:** Precise reduction of the voltage to 5V for the DVR without noise interference.
* **Signal integrity:** Use of an active booster to prevent signal loss when splitting the signal between the monitor and the DVR.
* **Ergonomics:** Mechanical integration of all components into a compact pistol grip.
